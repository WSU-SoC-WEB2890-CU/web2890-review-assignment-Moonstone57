My Specializations

UX Design

I am passionate about UX design. Designing with your users in mind is crucial for successful websites. When it comes to UX design it is important to conduct user research to understand the needs of your user base. Utilizing user research also helps create more effective prototypes. When these techniques are properly implemented into the design process of websites, it ensures that your website is designed based on data and not preconceived notions about your users.

Web Accessibility

The accessibility of webistes is indispensable. Many users are impacted when accessibility is not prioritized because they cannot actually complete their goals using the website. When websites are more user friendly and intuitive, all users benefit. It is important to evalutate the frustration points of websites and inovate accordingly. I believe it is important to ensure that all users, regardless of their background, can access your website.

Front-end Programming

Programming is the tool that creates the tangible website. I design optimized websites using coding languages such as: HTML, CSS, SASS, and JavaScript. Having the technical and problem solving skills required to code a website is important to the functionality of the website.

Project Highlight

I created a website called Aland Accelerando for users to sign up for music lessons. The purpose of this website was to allow users to look through many different teachers from one website.

I started this project by conducting user research. I discovered that my users wanted flexible, customizable lessons. I created a persona based on this called "Olivia Miller". This user research also helped me come up with the idea of packages. All the packages are suitable for begineers they just determine the style of lessons.
I created wireframes based on my user reached. I designed wireframes for both mobile and desktop. I did low fidelity wireframes to get the general layout, so that I could create a style guide. My color choices consisted of purples and greens. I wanted a warm but fun color scheme.
I created working prototypes using Figma. I made sure that hover effects and other functional aspects were working. It was important to me that the different user flows I wanted were possible to do through the prototype.
Using the prototypes I created in Figma I conducted usability testing to determine what changes and improvements needed to be made. This was very product and highlighted issues, such as not enough detail on the home page, that I could improve upon.
